mm-action

A fully-styled inline text link with optional icon.

Example HTML Editor

<!-- Text Example --> <mm-action> <label>+ Add Item</label> </mm-action> <!-- Text with Icon Example --> <mm-action unresolved> <mm-icon type="favorite" width="16" height="16" shapeRendering="auto" optimize="true"></mm-icon> <label>Awesome</label> </mm-action>

Example Output

Attributes

Attribute Options
href : String (Optional) Default value: none The link to use (if not using a click handler) none
underline : Boolean (Optional) Default value: none Underline action text none
