mm-collection

A simple collection wrapper component useful for 2 way binding to lists of data

Example HTML Editor

<!-- note this example is intended to show how to use a collection with a data adapter, the example adapter does not exist --> <template is="dom-bind"> <mm-collection id="searchResults" data="{{dataList}}" auto> </mm-collection> <template is="dom-repeat" items="{{datalist}}"> <div>{{item.name}}</div> </template> </template>
