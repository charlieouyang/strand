mm-grid-column

Provides configuration for data fields in mm-grid.

Example HTML Editor

<mm-grid id="mmTestGrid" unresolved style="width: 100%"> <mm-grid-column field="first_name" width="20%" resize sort>First Name</mm-grid-column> <mm-grid-column field="last_name" width="20%" resize sort align="right">Last Name</mm-grid-column> <mm-grid-column field="email" width="60%" sort>Email</mm-grid-column> </mm-grid> <script> var grid = document.querySelector("#mmTestGrid"), data = []; for(var i=0; i<50; i++){ data.push({ first_name: "First", last_name: "Last", email: "text@example.com" }); } window.addEventListener("polymer-ready", function() { grid.data = data; }); </script>

Example Output

First Name Last Name Email

Attributes

Attribute Options
field : String The data field/key to render into this column. none
label : String (Optional) The label for this column - optionally put the label into the HTML Content. none
width : String (Optional) The width of this column in percent or pixels. none
minWidth : Integer (Optional) The minimum width of the column in pixels when resizing none
resize : Boolean (Optional) Enable dynamic resizing for this column none
sort : Boolean (Optional) Enable sort events / indicators for this column none
sortField : String (Optional) Default value: column 'field' value The field to sort on - dispatched with 'column-sort' event none
sortOrder : Integer (Optional) Default value: 0 The sort direction - Ascending/None/Descending represented as an integer by 1 / 0 / -1 '1''0''-1'
align : String (Optional) Default value: left Align column content to left of right 'left''right'

Events

Event Event Detail
Type : 'column-sort' field : String The sortField / field to sort val : Integer The sort order
Type : 'column-resize-start' field : String The column's field val : Integer The column's offsetLeft value
Type : 'column-resize' field : String The column's field val : Integer The desired resize width value
Type : 'column-resize-end' field : String The column's field val : Integer The actual resized width value
