Public Properties

Properties Options
rules : Object An object containing a set of named functions used for validation. Method signature is (String char, String input, Object group, HTMLInputElement field) where char is the latest character a user has typed, and input is the full string the user has entered for a given group. none
restrict : Object An object containing a set of regexes used to test input. If input does not test true, it is not allowed in the field. none
placeholder : string Placeholder value. Should contain separator chars as well so it can be properly chunked into individual inputs. none
