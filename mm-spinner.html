mm-spinner

An animated spinning graphic which can be dispalyed during indeterminite loading states.

Example HTML Editor

<mm-spinner radius="9" id="sp1" fillColor="#333333"></mm-spinner> <mm-spinner radius="16" innerRadius="0.8" lineWidth="0.2" id="sp2" fillColor="#333333"></mm-spinner> <mm-spinner radius="24" innerRadius="5" lineWidth="10" lineWeight="1.5" id="sp3" fillColor="#333333"></mm-spinner> <mm-spinner radius="32" id="sp4" fillColor="#333333"></mm-spinner>

Example Output

Attributes

Attribute Options
radius : Number (Optional) Default value: 9 The radius of the spinner (which is always circular). As expected, radius of 9 yeilds an 18px circle. none
innerRadius : Number (Optional) Default value: 0.5 The inner radius of the spinner from where the line tick starts drawing. Values greater than 1 are pixel based - less than 1 is proportional to radius (ie. 0.5 == 50% of radius) none
numTicks : Number (Optional) Default value: 12 The number of ticks (lines) which compose the circumference of the spinner. Defaults to 12 for a 'standard' analog clock look. none
speed : Number (Optional) Default value: 1.33 Spinner speed in cycles per second. none
lineWidth : Number (Optional) Default value: 0.5 The width of each tick (line) which compose the circumference of the spinner. Values greater than 1 are pixel based - less than 1 is proportional to radius (ie. 0.5 == 50% of radius) none
lineWeight : Number (Optional) Default value: 0.5 The thickness of each tick (line) which compose the circumference of the spinner. Values greater than 1 are pixel based - less than 1 is proportional to the square root of radius (ie. 0.5 == sqrt(radius) * 0.5) none
fillColor : String (Optional) Default value: #ffffff The hex color number to fill each tick (line) which compose the circumference of the spinner. Expects a 7 character string (i.e. '#ffffff'). none
bgColor : String (Optional) Default value: null The hex color number to fill the background of the spinner. The default is null to leave the background transparent. none

Public Methods

Method Arguements
start() Instructs the spinner to begin animating. none
stop() Instructs the spinner to stop animating, and resets the spinners animation state to default. none
