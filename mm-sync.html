mm-sync

High level AJAX Component

Example HTML Editor

<template is="dom-bind"> <mm-sync id="search" endpoint="https://api.github.com/search/code?q=polymer+in:file+language:js+repo:polymer/polymer" data="{{data}}" auto> </mm-sync> <template is="dom-repeat" items="{{data.items}}"> <span>{{item.name}}</span> </template> </template>

default : {noop:function(){}}
options : none
attribute : api-marshaller

Keys in the object should be functions and should match names passed either in sync options or to response/requestMarshaller. In both cases the keyed method is called with the local data and it is expected to return the proper format for the API.
