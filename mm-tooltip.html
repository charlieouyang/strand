mm-tooltip

A custom styled tooltip which accepts variable and partially styleable content. IMPORTANT: This component requires template tags around the content you would like to insert.

Example HTML Editor

<!-- You must use a template tag to populate the tooltip --> <!-- Default Example --> <mm-tooltip> <mm-icon type="info" width="14" height="14" primaryColor="#666666"></mm-icon> <template> <label>Hi there, I'm a tooltip.</label> </template> </mm-tooltip> <!-- User dismissable example --> <mm-tooltip tipWidth="200" auto="false"> <template> <label><span class="bold">Hi there: </span>you'll have to click the x to dismiss me. Here's a <a href="http://www.example.com" target="_blank">link</a>, too!</label> </template> <mm-action underline="true"><label>Awesome</label></mm-action> </mm-tooltip>

Example Output

Attributes

Attribute Options
tipWidth : int (Optional) Default value: Defaults to content width The desired width of the tooltip. none
auto : Boolean (Optional) Default value: true Instructs the tooltip to stay visible even after the user has rolled out of the target area. 'true''false'
valign : String (Optional) Default value: top Where the tooltip is positioned relative to it's target area. 'top''bottom'

Public Methods

Method Arguements
open() Instructs the tooltip to be displayed. none
close() Instructs the tooltip to be hidden. none

Public Properties

Properties Options
tipWidth : int Default value: Defaults to content width The desired width of the tooltip. none
auto : Boolean Default value: Defaults to content width Instructs the tooltip to stay visible even after the user has rolled out of the target area. none
valign : String Default value: bottom Instructs the tooltip where to render it's arrow (pointer) indicator, relative to it's target area. none
